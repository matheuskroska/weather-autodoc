## weather-autodoc

![](https://i.imgur.com/9hurVaz.png)

teste para a vaga de desenvolvedor frontend

## instruções

utilizar o env.example deste repositório, adicionar as chaves do firebase e renomear para .env

* npm run start - roda o projeto

## tech

* Redux + Redux Toolkit
* Firebase Firestore
* Styled Components
* Jest
